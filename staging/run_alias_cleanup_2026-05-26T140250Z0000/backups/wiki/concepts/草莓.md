# 草莓

## Definition

草莓是一种常见的浆果类水果，颜色鲜红，味道酸甜，香气明显，适合直接食用和用于制作甜点、饮品、果酱等，富含维生素C但耐储存性差。

## Key Claims

- 草莓是一种常见的浆果类水果，颜色鲜红，味道酸甜，香气明显。它既适合直接食用，也常用于制作甜点、饮品、果酱和水果沙拉。 (`clm_src_0c4d4ef1c0eb_llm_001`, `src_0c4d4ef1c0eb`, `line:5;section:1. 概述;paragraph:1`)
- 草莓含有较丰富的维生素 C。 (`clm_src_0c4d4ef1c0eb_llm_002`, `src_0c4d4ef1c0eb`, `line:21;section:2.1 维生素 C;paragraph:5`)
- 在常见水果中，草莓的维生素 C 表现较好，是一种适合作为日常补充的水果。 (`clm_src_0c4d4ef1c0eb_llm_003`, `src_0c4d4ef1c0eb`, `line:23;section:2.1 维生素 C;paragraph:6`)
- 草莓中含有一定膳食纤维，有助于增加饱腹感。 (`clm_src_0c4d4ef1c0eb_llm_004`, `src_0c4d4ef1c0eb`, `line:27;section:2.2 膳食纤维;paragraph:7`)
- 草莓鲜红的颜色与其中的天然色素和植物化合物有关。 (`clm_src_0c4d4ef1c0eb_llm_005`, `src_0c4d4ef1c0eb`, `line:31;section:2.3 植物化合物;paragraph:8`)
- 草莓品种较多，不同品种在大小、甜度、酸度、香气和口感上会有差异。 (`clm_src_0c4d4ef1c0eb_llm_006`, `src_0c4d4ef1c0eb`, `line:35;section:3. 常见草莓类型;paragraph:9`)
- 草莓表面比较娇嫩，清洗和保存时都需要注意。 (`clm_src_0c4d4ef1c0eb_llm_007`, `src_0c4d4ef1c0eb`, `line:71;section:5. 清洗与保存方法;paragraph:17`)
- 草莓建议吃之前再清洗，不要提前大量清洗后存放。 (`clm_src_0c4d4ef1c0eb_llm_008`, `src_0c4d4ef1c0eb`, `line:75;section:5.1 清洗方法;paragraph:18`)

- 草莓果肉较软，受到挤压后容易破损、出水、变质。 (`clm_src_0c4d4ef1c0eb_llm_010`, `src_0c4d4ef1c0eb`, `line:90;section:5.3 避免挤压和久放;paragraph:22`)
- 草莓适合大多数人群食用，尤其适合喜欢酸甜口感、希望增加水果种类的人。 (`clm_src_0c4d4ef1c0eb_llm_011`, `src_0c4d4ef1c0eb`, `line:94;section:6. 适合人群与注意事项;paragraph:23`)
- 对草莓过敏的人不应食用。 (`clm_src_0c4d4ef1c0eb_llm_012`, `src_0c4d4ef1c0eb`, `line:98;section:6. 适合人群与注意事项;paragraph:25`)
